# Rebalancing trials

The current target baseline is a daily, survivor-first Hyperliquid strategy using an `age_ramp` signal and capped waterfall sizing. Its stored result is **458.89% CAGR, 2.92 Sharpe, and -10.69% max drawdown**; the target uses a later end date and a different initial capital, so the historical figures below are directional rather than directly comparable.

The table keeps distinct selection, weighting, allocation, and rebalancing approaches. Duplicate parameter searches and robustness reruns are omitted unless they represent a materially different portfolio-construction rule.
